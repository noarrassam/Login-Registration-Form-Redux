# Login & Registration Form - React Redux Version

## Description

It is a React Redux application that allows a user to register, login, edit, and view user's pages.

## Used

- HTML

- CSS

- Bootstrap-CDN

- React Redux

- Controlled Components

- Routes

- Forms

- React Hooks: useState, useContext

- Lazy Loading
